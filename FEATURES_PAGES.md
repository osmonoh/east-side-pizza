# FEATURES + PAGES

## FEATURE CATEGORIES

All features can be placed into one of these. So this is what the app will essentially be about:

-   User
-   Menu
-   Cart
-   Order

## PAGES

-   Homepage -> '/'
-   Pizza menu -> '/menu'
-   Cart -> '/cart'
-   Placing a new order -> '/order/new'
-   Looking up an order -> '/order/:orderID'
